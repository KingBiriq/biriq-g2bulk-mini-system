# Biriq G2Bulk Mini System

Mini system-kan wuxuu leeyahay:
- Backend Node.js + Express
- Frontend HTML/CSS/JS
- API key-ga wuxuu ku jiraa backend `.env`, frontend-ka lama tusayo
- Games/products API sync
- Player ID check
- Test order endpoint

## Sida loo shido

```bash
cd biriq-g2bulk-mini-system
npm install
copy .env.example .env
npm run dev
```

Kadib fur:
```text
http://localhost:3000
```

## Important

`.env` geli API key-gaaga:

```env
G2BULK_API_KEY=your_key_here
```

Ha gelin API key-ga frontend JavaScript.
